## Project Difficulty Level
Hard

---

## Project Description
This project is a classic two-player Tic-Tac-Toe game developed using HTML, CSS, and JavaScript. It allows two players to take turns on the same device, checks for winning or tie conditions, and displays the result. The project helps in practicing front-end development concepts such as layout design, styling, and JavaScript game logic.

---

## Project Objectives
- Create a user-friendly interface using HTML and CSS
- Implement game logic using JavaScript
- Allow two players to take turns
- Detect win and tie conditions
- Make the game responsive for desktop and mobile devices

---

## Project Features
- 3×3 Tic-Tac-Toe game board
- Displays current player’s turn
- Automatically detects winner or tie
- Highlights the winning combination
- Prevents moves after game completion
- Reset button to start a new game
- Responsive design for different screen sizes
